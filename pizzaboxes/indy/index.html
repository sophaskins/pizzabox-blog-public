External Storage none
Serial Ports 2x mini-DIN 8
Keyboard PS/2 (mini-DIN 6)

Networking 10Base-T Ethernet
  AUI Ethernet
  ISDN (RJ-45 Connector)
Audio 2x 1/8 ″ audio input (microphone and line-in)

  S-Video (mini-DIN 4) input
  IndyCam (bundled webcam) input
  3D glasses output
